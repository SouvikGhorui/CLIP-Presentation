Symmetric
Contrastive Loss

CLIP trains with a symmetric objective: both image→text and text→image contrastive losses are computed and averaged.

Image → Text Direction

For each image, find its matching text among all $N$ texts:

$$\mathcal{L}_i^{I \to T} = -\log \frac{e^{s_{ii}/\tau}}{\sum_{k} e^{s_{ik}/\tau}}$$

Text → Image Direction

For each text, find its matching image among all $N$ images:

$$\mathcal{L}_j^{T \to I} = -\log \frac{e^{s_{jj}/\tau}}{\sum_{k} e^{s_{kj}/\tau}}$$

Image → Text

softmax across rows

Text → Image

softmax across columns

TOTAL CLIP LOSS

$$\mathcal{L} = \frac{1}{2}\left(\mathcal{L}^{I \to T} + \mathcal{L}^{T \to I}\right)$$
