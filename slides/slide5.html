Zero-Shot
Classification

The true power of CLIP: classify images into any set of categories without any task-specific training data.

1. Create Text Prompts

For each candidate class $c$, construct a text prompt:

"A photo of a {class name}"

Encode each prompt: $\mathbf{T}_c = \text{TextEncoder}(\text{prompt}_c)$

2. Encode the Image

Pass the query image through the image encoder:

$$\mathbf{I}_x = \text{ImageEncoder}(x)$$

3. Compare via Cosine Similarity

Compute similarity between the image and every class prompt, then apply softmax:

ZERO-SHOT PREDICTION

$$p(y{=}c \mid x) = \frac{\exp\!\left(\mathbf{I}_x^{\top}\mathbf{T}_c\, /\, \tau\right)}{\displaystyle\sum_{c'}\exp\!\left(\mathbf{I}_x^{\top}\mathbf{T}_{c'}\, /\, \tau\right)}$$

Predict: $\hat{y} = \arg\max_c \; p(y{=}c \mid x)$
